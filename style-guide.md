# Character Counter - Style Guide (스타일 가이드)

## 1. Neutral (무채색 계열)
주로 배경, 텍스트, 경계선 등에 사용되는 기본 색상들입니다.

| 색상 이름 | Hex 코드 | 설명 |
| :--- | :--- | :--- |
| **Neutral 900** | `#12131A` | 가장 어두운 텍스트 또는 배경색 |
| **Neutral 800** | `#21222C` | 카드 배경 등 어두운 영역 |
| **Neutral 700** | `#2A2B37` | 보조적인 어두운 배경 |
| **Neutral 600** | `#404254` | 보조 텍스트 또는 아이콘 |
| **Neutral 200** | `#E4E4EF` | 밝은 배경의 경계선이나 테두리 |
| **Neutral 100** | `#F2F2F7` | 매우 밝은 보조 배경 |
| **Neutral 0** | `#FFFFFF` | 순수 흰색 (기본 배경 또는 텍스트) |

## 2. Purple (디자인상 Blue로 명명됨)
포인트 색상(Primary Color)으로 사용되는 보라색 계열입니다. 디자인 가이드에는 이름이 Blue로 되어 있으니 주의하세요!

| 색상 이름 | Hex 코드 | 설명 |
| :--- | :--- | :--- |
| **Blue 400** | `#D3A0FA` | 밝은 보라색 (호버 상태 등) |
| **Blue 500** | `#C27CF8` | 메인 포인트 보라색 |

## 4. Typography (타이포그래피 - 글꼴 가이드)

프로젝트의 전체적인 분위기를 결정하는 글꼴 설정입니다. 모든 텍스트에는 **DM Sans** 글꼴을 사용합니다.

| 분류 | 굵기 (Weight) | 크기 (Size) | 줄 간격 (Line Height) | 자간 (Letter Spacing) |
| :--- | :--- | :--- | :--- | :--- |
| **Text Preset 1** | Bold (700) | 64px | 100% | -1px |
| **Text Preset 1 (Mobile)** | Bold (700) | 40px | 100% | -1px |
| **Text Preset 2** | SemiBold (600) | 24px | 130% | -1px |
| **Text Preset 3** | Regular (400) | 20px | 140% | -0.6px |
| **Text Preset 4** | Regular (400) | 16px | 130% | -0.6px |

### 💡 용어 설명 (초보자를 위한 가이드)
- **Font Size (크기):** 글자의 크기입니다. 숫자가 클수록 글자가 화면에서 크게 보여요.
- **Line Height (줄 간격):** 글자 줄 사이의 간격(키)이에요. 100%면 글자 크기와 딱 맞고, 숫자가 커질수록 줄 사이가 넓어져서 눈이 편해져요.
- **Letter Spacing (자간):** 글자와 글자 사이의 가로 간격이에요. 마이너스(-) 값이면 글자들이 서로 더 친하게 붙어있는 느낌을 줍니다.
- **Font Weight (굵기):** 글자의 두께예요. Bold는 아주 굵고, Regular는 평범한 굵기를 말해요.

## 5. Spacing (간격 - 여백 가이드)

요소들 사이의 거리나 내부 여백(Padding)을 설정할 때 사용하는 수치들입니다.

| 이름 (Name) | 픽셀 (Pixels) | 설명 |
| :--- | :--- | :--- |
| **spacing-0** | 0px | 여백 없음 |
| **spacing-025** | 2px | 아주 미세한 간격 |
| **spacing-050** | 4px | 좁은 간격 |
| **spacing-075** | 6px | 디자인 포인트 간격 |
| **spacing-100** | 8px | 기본 작은 간격 |
| **spacing-150** | 12px | 중간-작은 간격 |
| **spacing-200** | 16px | 일반적인 내부 여백 (자주 사용) |
| **spacing-250** | 20px | 중간 간격 |
| **spacing-300** | 24px | 넉넉한 간격 |
| **spacing-400** | 32px | 큰 간격 |
| **spacing-500** | 40px | 아주 큰 간격 |
| **spacing-600** | 48px | 섹션 사이의 여백 |
| **spacing-800** | 64px | 큰 섹션 사이의 여백 |
| **spacing-1000** | 80px | 레이아웃의 가장 큰 여백 |

### 💡 여백(Spacing)이 중요한 이유
- **숨 쉴 공간:** 요소들이 너무 다닥다닥 붙어 있으면 답답해 보여요. 적절한 여백은 사용자에게 시각적인 편안함(숨 쉴 공간)을 줍니다.
- **일관성:** 정해진 주소(`spacing-200` 등)를 사용하면 여러 페이지를 만들어도 디자인이 통일감 있게 유지된답니다!

## 6. Border Radius (모서리 둥글기 가이드)

카드나 버튼 등 요소의 모서리를 얼마나 둥글게 깎을지 결정하는 수치입니다.

| 이름 (Name) | 픽셀 (Pixels) | 설명 |
| :--- | :--- | :--- |
| **radius-0** | 0px | 깎임 없음 (완전한 직각형) |
| **radius-4** | 4px | 살짝 부드러운 느낌 |
| **radius-6** | 6px | 디자인 보조 둥글기 |
| **radius-8** | 8px | 기본 부드러운 모서리 |
| **radius-10** | 10px | 중간 정도의 둥글기 |
| **radius-12** | 12px | 눈에 띄는 부드러운 모서리 |
| **radius-16** | 16px | 카드와 같은 큰 요소에 적합 |
| **radius-20** | 20px | 둥근 형태가 강조된 스타일 |
| **radius-24** | 24px | 강한 곡선 스타일 |
| **radius-full** | 999px | 완전히 둥근 형태 (알약/원 모양) |

### 💡 둥글기(Radius)가 주는 느낌
- **딱딱함 vs 부드러움:** 모서리가 날카로운 직각(`radius-0`)이면 전문적이고 단단한 느낌을 주고, 둥글어질수록 친근하고 부드러운 느낌을 줍니다.
- **완성도의 차이:** 모든 버튼이나 카드의 모서리 값을 똑같이 맞추는 것만으로도 웹사이트의 완성도가 훨씬 높아 보인답니다!
